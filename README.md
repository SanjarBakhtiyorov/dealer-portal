# NormDealer — Нормативная модель дилера

Streamlit-приложение для расчёта нормативных расходов дилеров на основе данных стока.

## Возможности

- 📈 **Дашборд** — ключевые метрики, расходы по категориям, статус нормативов
- 📋 **Детализация** — разбивка по каждой статье расходов с формулой расчёта
- 📂 **Загрузка стока** — загрузка Excel файла дилера, автоматический пересчёт
- ⊞ **Все дилеры** — сводная таблица по всем дилерам с экспортом
- ⚙️ **Допущения** — редактируемые оклады, налоги, ставки аренды — всё пересчитывается мгновенно

## 🚀 Деплой на Streamlit Cloud (бесплатно)

### Шаг 1 — Загрузите код на GitHub

```bash
# Создайте новый репозиторий на github.com, затем:
git init
git add .
git commit -m "Initial commit — NormDealer portal"
git remote add origin https://github.com/ВАШ_USERNAME/dealer-portal.git
git push -u origin main
```

### Шаг 2 — Подключите Streamlit Cloud

1. Перейдите на [share.streamlit.io](https://share.streamlit.io)
2. Войдите через GitHub аккаунт
3. Нажмите **"New app"**
4. Выберите ваш репозиторий и ветку `main`
5. Убедитесь, что **Main file path** = `app.py`
6. Нажмите **"Deploy"**

Через ~2 минуты ваше приложение будет доступно по адресу:
```
https://ВАШ_USERNAME-dealer-portal-app-XXXXX.streamlit.app
```

## Как добавить новых дилеров

В файле `app.py` найдите словарь `DEALERS` и добавьте новую запись:

```python
DEALERS = {
    "Имя Дилера": {
        "sap":       "1000001234",   # SAP код
        "rev":       500000,          # Месячная выручка в USD
        "pallets":   3000,            # Паллетомест
        "area":      1200.0,          # Площадь склада в м²
        "gross_pct": 0.058,           # Доля валовой прибыли (5.8%)
    },
    # ... остальные дилеры
}
```

После загрузки реального Excel файла дилера через вкладку "Загрузить данные стока", значения `pallets` и `area` обновляются автоматически.

## Формат файла для загрузки стока

| Колонка | Тип | Обязательно |
|---------|-----|-------------|
| Артикул | текст | ✓ Да |
| Остаток (шт) | число | ✓ Да |
| Шт/поддон | число | ✓ Да |
| Описание | текст | Нет |
| Объем ед. (м³) | число | Нет |

Шаблон для заполнения можно скачать прямо в приложении на вкладке "Загрузить данные стока".
